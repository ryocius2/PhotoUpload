[ ] 3. HARDEN THE PI (prevent SD card corruption & WiFi drops)
      a) Mount USB drive for photo storage (all writes go to USB, not SD):
         - Plug in the SanDisk USB drive
         - sudo mkdir -p /mnt/photos
         - sudo blkid   (find the USB drive, e.g. /dev/sda1)
         - sudo mkfs.ext4 /dev/sda1   (format it — only need to do once!)
         - sudo mount /dev/sda1 /mnt/photos
         - sudo chown pi:pi /mnt/photos
         - Make it permanent: sudo nano /etc/fstab
           Add this line:
           /dev/sda1 /mnt/photos ext4 defaults,noatime 0 2
         - Update .env: nano ~/PhotoUpload/.env
           Change UPLOAD_FOLDER to /mnt/photos

      b) Disable WiFi power management:
         - sudo nano /etc/NetworkManager/conf.d/wifi-powersave-off.conf
           Add:
           [connection]
           wifi.powersave = 2

      c) WiFi watchdog (auto-recovers dropped WiFi):
         - sudo nano /usr/local/bin/wifi-watchdog.sh
           Add:
           #!/bin/bash
           if ! ping -c 1 -W 5 8.8.8.8 > /dev/null 2>&1; then
               nmcli radio wifi off
               sleep 2
               nmcli radio wifi on
           fi
         - sudo chmod +x /usr/local/bin/wifi-watchdog.sh
         - sudo crontab -e
           Add: */5 * * * * /usr/local/bin/wifi-watchdog.sh

      d) Set root filesystem to read-only (optional, extra protection):
         - sudo raspi-config -> Performance -> Overlay File System -> Enable
         - This makes the SD card read-only, only /mnt/photos is writable
         - NOTE: to make changes later you need to disable this first
